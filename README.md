# Time Series Stock Forecasting

This project explores time series models for stock price forecasting.

## Methods
- (e.g.) ARIMA / ARMA
- (e.g.) GARCH for volatility
- Baselines and evaluation

## Project Structure
- `src/`: source code
- `notebooks/`: experiments
- `data/`: data (ignored by git)
- `reports/`: reports / notes
- `figures/`: plots

```bash
pip install -r requirements.txt
# python src/main.py
